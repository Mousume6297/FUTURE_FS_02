# 🚀 ClientHub CRM

A modern Customer Relationship Management (CRM) system built using React.
This project helps manage leads, staff, and user profiles with a clean and professional UI.

---

## 📌 Features

* 📊 Dashboard with analytics
* 👥 Lead Management (Add, Edit, Delete)
* 🧑‍💼 Staff Management
* 🔐 Role-Based Access (Admin, Manager, HR, Viewer)
* 👤 Profile Management with Avatar Upload
* 🔍 Search & Filter Leads
* 📈 Growth Indicators on Dashboard
* 🎨 Clean and Professional UI

---

## 🛠️ Tech Stack

* Frontend: React.js
* Styling: CSS
* State Management: useState, useEffect
* Storage: LocalStorage
* Icons: React Icons

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/clienthub-crm.git
```

2. Navigate to project folder

```bash
cd clienthub-crm
```

3. Install dependencies

```bash
npm install
```

4. Run the project

```bash
npm start
```
